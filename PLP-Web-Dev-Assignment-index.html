Assignment Overview

This page is built using semantic HTML5 elements and focuses on clear hierarchy, screen-reader friendliness, and SEO-ready content.

About This Example

Key goals for this assignment:

  • Use meaningful semantic elements (header, nav, main, article, section, footer).
  • Follow a logical heading order (H1 > H2 > H3) to support screen readers and SEO.
  • Provide descriptive metadata and alt text so both humans and crawlers understand the content.

Accessibility & SEO Examples

On-Page SEO Basics

Search engines look for clear structure and meaningful content. Use a descriptive title, meta description, and headings that reflect the topic of each section. Avoid heading skipping (don't jump from H1 to H4 without intermediate levels) and provide textual content that answers user intent.

Accessibility Enhancements

Provide keyboard support (skip links), meaningful link text (avoid "click here"), alt text for images, and explicit form labels. Use landmark elements and ARIA attributes only when native semantics are insufficient.

Quick checklist

  • Document language: lang set on <html>.
  • Skip link present for keyboard users.
  • Logical heading order and descriptive headings.
  • Descriptive meta description for crawlers.
  • Accessible form controls with labels.
